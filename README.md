# Rusted-Firmware A Artifact

This repository automates the build of [Rusted Firmware A (RF-A)](https://git.trustedfirmware.org/RF-A/rusted-firmware-a) for the purpose of generating artifacts that can be used for integration tests.

## How to release new artifacts

Create a new tag and push it to upstream, e.g.:

```sh
git tag v0.1.0
git push origin v0.1.0
```
